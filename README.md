## Features

- Manage your own recipes.
- View other recipes.
- Comment on recipes.

## Installation

Clone the repository and run the following command.

```
make setup
```

Setup database connection in `.env` and run.

```
make update
```

Done!

## Development

Every time your pull in changes make sure to run.

```
make update
```
